### jquery.adaptive-slider
---
https://github.com/creative-punch/jQuery.adaptive-slider/

```js
$(function(){
  $('.adaptive-slider').adaptiveSlider({
    opacity: 0.7,
    normalizedTextColors: {
      light: '#f1f1f1',
      dark: '#222'
    }
  });
});
```
